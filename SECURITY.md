# Política de Seguridad

## Versiones Soportadas

Actualmente, solo la versión en producción (rama `main`) recibe actualizaciones de seguridad.

| Versión | Soportada          |
| ------- | ------------------ |
| 5.x     | :white_check_mark: |
| < 5.0   | :x:                |

## Reportar una Vulnerabilidad

Si encuentras un fallo de seguridad (ej. inyección SQL, fallo en RLS, exposición de datos), por favor **NO abras un Issue público**.

Envía un correo a: hola@campa.dev

Trataremos el reporte con prioridad máxima.
